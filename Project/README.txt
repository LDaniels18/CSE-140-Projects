In order to use this Single Cycle CPU

You must enter "Make" in the terminal, this will compile the project files
Then enter "./output" in order to view the output

In order to change the set of instructions used, simple save a textfile to the project repository and change the filename within the Single_Cycle_MIPS_CPU.c file (will be labled)

In order to clean the project repository, simply enter "make clean" in the terminal
